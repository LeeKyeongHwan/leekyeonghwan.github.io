[후기] 12가지 명령어로 배우는 Git
Oct 26, 2019

zerocho님의 유튜브 무료강의인 12가지 명령어로 배우는 Git을 보고난 후기

함부로 이런 후기를 남기는게 실례가 될 수도 있지만, 상당히 좋은 강의라고 생각하기에 기분좋게 후기를 남긴다.

일단 동영상의 평균적인 길이가 10분~20분 사이여서, 출퇴근 시간에 가볍게 한 두편씩 보기 좋았다.

기본적인 git의 기능은 알고 있고, 실무에서 사용한 경험이 있었기 때문에 이해하기에 크게 무리가 없었으며,
초보자나 기본적인 기능만 아는 분들이 보면 굉장히 유익할 것 같다.

총 12가지의 git의 기본적인 기능을 설명한다.

  1. status
  2. add
  3. commit
  4. remote
  5. push
  6. pull
  7. reset
  8. revert
  9. branch
  10. merge
  11. rebase
  12. cherry-pick

위의 명령어 외에도 tag, stash, checkout, fetch 등에 대해서도 간략하게 설명한다.
실제 git을 사용할 때를 가정, 실습하여 명령어를 순차적으로 설명해주는게 이해하는데 도움이 많이 되었다.
최종적으로는 git을 사용하는 전략인 git flow에 대해서도 설명하는데 이 부분이 특히 흥미로웠다.
git flow에 대해 들어만 봤지 사용을 해보지는 못했는데, 어렵지않게 이해하고 장점에 대해 파악할 수 있었다.

JavaScript this 정리
Oct 09, 2019

자바스크립트에서 this는 호출되는 방식에 따라 달라지기 때문에, 이해하기 너무 어렵다.
그래서 아예 대표적인 몇 가지를 그냥 외워버리기로 했다.

  1. 자유함수 호출
    가장 기본적인 경우, 별다른 조건이 없을 때에 this는 전역객체를 바라본다.

    browser의 전역객체는 window
    node의 전역객체는 global

  2. 생성자 호출
    new 키워드를 이용한 생성자 호출시에 함수 내부 this는 새로운 객체를 바라본다.

  3. Object의 Method 호출
    Object의 Method를 호출했을 때엔 this는 Object를 가리킨다.

  4. Arrow Function 호출
    ES6에서 추가된 Arrow Function의 경우, 생성된 시점의 주변 스코프의 this를 갖게된다

  5. call, apply, bind로 함수 호출
    각각 함수를 호출하고, 정의하는 Function의 메소드이다. 첫번째 인자를 통해 this를 변경할 수 있다.
